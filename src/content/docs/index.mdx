## What you'll find here

These wikis are partner-facing reference material that sits between the technical primary literature and the day-to-day analytic tools CGIAR programmes use. If you're writing a climate rationale for a funding proposal, designing a research-for-development programme, or just need a citable explanation of *why* the Atlas (or another tool) makes certain methodological choices — start here.

## About this site

This is the *staging and testing* repository for wikis that the CGIAR Climate Data Hub team publishes. Authoring happens here in plain Markdown (with optional MDX for richer components); the site is built with [Astro Starlight](https://starlight.astro.build/) and deployed to GitHub Pages on every push to `main`.

Source of truth varies per wiki — some are drafted in their home repository and synced here; others are authored directly here. Each page's `frontmatter` makes the source explicit.
